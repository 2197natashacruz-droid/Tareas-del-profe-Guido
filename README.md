# Introducción a la Programación -- Hello World

Este repositorio muestra ejemplos básicos del clásico programa **Hello
World** en distintos lenguajes de programación.\
El objetivo es entender conceptos fundamentales como **qué hace un
programa**, **qué es la sintaxis** y **cómo se escribe código en
diferentes lenguajes**.

------------------------------------------------------------------------

## 1. ¿Qué hace el programa "Hello World"?

El programa **Hello World** es normalmente el primer programa que se
escribe cuando se aprende un lenguaje de programación.

Su función es simplemente **mostrar un mensaje en la pantalla**,
generalmente:

    Hello, World!

### ¿Para qué se usa?

-   Verificar que el lenguaje de programación está instalado
    correctamente.
-   Aprender la estructura básica de un programa.
-   Comprobar que el código se puede ejecutar sin errores.

------------------------------------------------------------------------

## 2. ¿Qué significa "sintaxis" en programación?

La **sintaxis** es el conjunto de **reglas que indican cómo se debe
escribir el código** en un lenguaje de programación.

Cada lenguaje tiene su propia forma de escribir las instrucciones.
Aunque el resultado sea el mismo, la forma de escribirlo cambia.

Por ejemplo, imprimir un mensaje en pantalla se escribe diferente según
el lenguaje.

------------------------------------------------------------------------

## 3. Ejemplos de "Hello World"

### Python

``` python
print("Hello, World!")
```

Python es conocido por tener una sintaxis muy simple y fácil de leer.

------------------------------------------------------------------------

### JavaScript

``` javascript
console.log("Hello, World!");
```

Este código muestra el mensaje en la **consola del navegador** o en
entornos como Node.js.

------------------------------------------------------------------------

### Java

``` java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

Java requiere una estructura más completa porque es un lenguaje
**orientado a objetos**.

------------------------------------------------------------------------

## 4. ¿Qué es Java?

**Java** es un lenguaje de programación ampliamente utilizado para
desarrollar:

-   Aplicaciones empresariales
-   Aplicaciones móviles (Android)
-   Software de escritorio
-   Sistemas web

Es conocido por su filosofía:

> *Write Once, Run Anywhere*\
> (Escribe una vez, ejecútalo en cualquier lugar)

Esto significa que los programas de Java pueden ejecutarse en diferentes
sistemas operativos.
